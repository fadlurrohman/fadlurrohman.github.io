Breast Lesion Detection & Classification on Mammograms

Muhammad Fadlurrohman, Farchan Hakim Raswa, Bach-Tung Pham, Jia-Ching Wang, DLMSL Lab.

  • Code
  • Youtube
    • Developed a deep learning–based object detection and classification system to identify breast lesions from mammogram images. The project focused on detecting and classifying breast masses and calcifications into benign and malignant categories to support clinical decision-making. Implemented a bounding-box–based lesion localization pipeline with multi-class labels such as MASS_BENIGN, MASS_MALIGNANT, and CALC_BENIGN. Applied image preprocessing and normalization techniques to enhance lesion visibility and reduce background noise in mammography data. Trained and evaluated deep learning models for simultaneous lesion detection and classification, and visualized model predictions using overlaid bounding boxes and class labels to improve interpretability and validation.
